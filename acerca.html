🌦️ Sobre la aplicación

Este proyecto corresponde al desarrollo de una aplicación web del clima en su versión mejorada (M3), como continuación del módulo de frontend.

Su propósito es mostrar información del clima actual y el pronóstico de distintas ciudades, utilizando HTML5 semántico, Bootstrap para el diseño responsivo y SASS para la organización y personalización de estilos.

En esta versión se incorporó el uso de la metodología BEM para estructurar las clases CSS, además de variables y mixins en SASS que permiten un código más ordenado, reutilizable y mantenible.

La aplicación permite visualizar una lista de ciudades con información básica del clima y acceder a un detalle donde se muestran datos como temperatura, estado del clima y otras condiciones relevantes. También se mejoró la experiencia visual mediante el uso de efectos hover, diseño más consistente en tarjetas y navegación más clara entre páginas. Este proyecto representa una evolución hacia buenas prácticas de desarrollo frontend, enfocadas en la organización del código y la escalabilidad.
