🌦️ Acerca del Proyecto

Este proyecto corresponde al desarrollo de una aplicación web del clima en su versión inicial, realizada como finalización del módulo 2 de Frontend.

Su propósito es mostrar información del clima actual y el pronóstico semanal de distintas ciudades de interés, utilizando HTML5 semántico, Bootstrap para el diseño responsivo y CSS para la personalización de estilos.

La aplicación permite visualizar una lista de ciudades con información básica del clima y acceder a un detalle donde se muestran datos como temperatura, humedad, viento y pronóstico semanal.

Esta versión fue desarrollado sin el uso de JavaScript, dando prioridad a la ejecución de la estructura del contenido y navegación entre páginas.
